# mlviewer
Memory View & Edit PyQt program powered by Frida<br>

# Prerequisite
```
python > 3.8.0
```

# Usage
Run
```
git clone https://github.com/hackcatml/mlviewer

cd mlviewer

pip install -r requirements.txt

python main.py
```
